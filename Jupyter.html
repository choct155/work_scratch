Sharing¶

The Notebooks themselves are fairly lightweight JSON files. They can be opened and edited by anyone who has Jupyter Notebook (and the language-appropriate kernel). The real gem for static formats, however, is nbconvert. This functionality can convert dynamic Notebooks into static files in a number of formats, including the following:

  1. Standalone HTML and Reveal.js slide decks (which can be opened by anyone with a browser);
  2. LaTeX
  3. PDF
